InstallationÂ¶

Please note that since Python>=3.5 is required, all of the following commands, especially pip, also have to be the Python 3 compliant versions thereof. This might require that you run pip3 instead.

Install COMET as a package with pip:

pip install unbabel-comet

Inside your project you can now:

import comet

or run it with our command line interface:

comet --help
